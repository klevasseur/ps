\pmb{ Definition}: A function \(f\) defined on an interval $\mathcal{I}$ (possibly infinite) is \textit{ concave} if for all \(a, b\in \mathcal{I}\)
and \(\lambda \in (0,1)\), \(\lambda  f(a) + (1-\lambda )f(b)\leq f(\lambda  a+(1-\lambda )b)\).
